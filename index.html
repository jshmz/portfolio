LINE版AthletesPort

LINEのトークでコンディション管理ができるアプリを構築しました。LINEを利用することでアプリのインストールの手間をなくし、LINE Front-end Frameworkを利用することでログインの手間をなくしました。日々のコンディション情報を入力すると、グラフ表示され、データによって簡易的なアラートが表示されます。起床時心拍数や睡眠時間等を自動で登録するためOura ring APIと連携し、データを定期的に自動取得する仕組みも実装しました。

コンディションデータを
元にした上気道炎の予測

コンディショニング情報管理アプリのデータを用いて、上気道炎（風邪）の予測を行う機械学習モデルの構築を行いました。アスリートによってデータ入力項目がまちまちで、欠損値があること、また上気道炎と判定できるケースが極めて少ない不均衡データであったため、LightGBMでアンダーサンプリングを行ってモデル構築を行い、上気道炎の症状を数日前に70％程度検出可能な予測モデルを構築しました。

【言語】 Python
【フレームワーク等】 LightGBM, Pandas, Numpy, Jupyter Notebook

イベント用Webクイズアプリ

セントレア空港で行われた2000人規模のイベントでのWebクイズアプリの実装と運用を行いました。実際のクイズに参加したのは500名程度でした。WebSocketを用いてクイズの問題を参加者のスマートフォンにPushし、その問題にいち早く答えた人が勝ちとなるイベントでした。データベースには速度と構築の容易さを鑑み、SQLiteをインメモリで動作させ、速度を担保しました。

当時のイベントを紹介したブログ記事です

【言語】 PHP, JavaScript
【OS】 Ubuntu
【ミドルウェア等】 Node.js, Apache, SQLite, WebSocket Server
【フレームワーク等】 独自フレームワーク
